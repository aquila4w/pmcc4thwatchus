# PMCC 4th Watch — US District Website

Pentecostal Missionary Church of Christ (4th Watch) US District website built with Next.js, Payload CMS, TypeScript, and Tailwind CSS.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
